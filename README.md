========================================
电力负荷预测项目说明
========================================

一、项目简介
----------------------------------------
本项目基于深度学习方法，实现对电力系统短期负荷的预测。
通过融合历史负荷数据、气象数据以及时间特征，构建LSTM、RNN及BPNN模型，
对未来电力负荷进行预测，并对不同模型性能进行对比分析。

本项目为本科毕业设计配套代码。

----------------------------------------

二、项目结构
----------------------------------------
│── data/                数据目录（示例数据）
│── model/               训练好的模型参数
│── src/                 核心代码
│   ├── train_lstm.py    LSTM模型训练
│   ├── train_rnn.py     RNN模型训练
│   ├── train_bpnn.py    BPNN模型训练
│   ├── compare.py       模型对比与评估
│   ├── visualize.py     结果可视化
│
│── utils/               工具函数
│   ├── common.py        数据处理与特征工程
│   ├── log.py           日志工具
│
│── data/fig/            可视化结果图片
│── README.md           项目说明文件

----------------------------------------

三、运行环境

----------------------------------------

四、运行方法
----------------------------------------

1. 训练模型（以LSTM为例）
----------------------------------------
python src/train_lstm.py


2. 模型评估与对比
----------------------------------------
python src/compare.py


3. 结果可视化
----------------------------------------
python src/visualize.py


----------------------------------------

五、主要功能说明
----------------------------------------

1. 数据预处理
- 缺失值填充（插值 + 前后填充）
- 时间特征提取（小时、星期、周期编码）
- 节假日与季节特征构建
- 滞后特征与滑动统计特征

2. 模型训练
- LSTM（主模型）
- RNN（对照模型）
- BPNN（对照模型）

3. 模型评估
- RMSE
- MAE
- MAPE

4. 可视化输出
- 预测曲线对比图
- 误差对比图
- 指标对比图
